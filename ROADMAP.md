# Roadmap

## Fase 0 (completada)

- base Python modular,
- CLI MVP funcional,
- pipeline reproducible con recipe,
- tests iniciales,
- trazabilidad JSON.

## Fase 1

- robustecer detección ColorChecker24 en datasets reales,
- soportar IT8 completo,
- mejorar score de calidad e iluminación no uniforme.

## Fase 2

- endurecer motor ArgyllCMS:
  - presets por tipo de perfil,
  - validación cruzada automática,
  - reporte comparativo Argyll vs ajuste interno.

## Fase 3

- mayor determinismo inter-plataforma,
- benchmarks de consistencia colorimétrica,
- suites de regresión con dataset de patrimonio/documentación técnica.

## Fase 4

- empaquetado y distribución comunitaria,
- guías de contribución científica,
- GUI ligera opcional (post-estabilización del núcleo).
