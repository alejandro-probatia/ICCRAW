# ProbRAW 0.3.14

ProbRAW 0.3.14 estabiliza el punto de equilibrio alcanzado entre rendimiento
interactivo y precision de lectura en la GUI.

## Cambios Principales

- La preview interactiva actualiza muestras de histograma y niveles durante
  arrastres sin bloquear la interfaz, y mantiene el refresco exacto cuando la
  interaccion se estabiliza.
- Los controles de curva tonal mantienen visible su histograma mientras se
  arrastran puntos o el rango `Negro curva`/`Blanco curva`.
- El visor incorpora recorte visual, nivelado por referencia horizontal o
  vertical, rotacion fraccional y reproyeccion del recorte al cambiar de
  resolucion o pedir pixeles reales.
- El nuevo menu `Editar` aporta deshacer, rehacer, eliminar ajustes y atajos
  `Ctrl+Z`/`Ctrl+Y`.

## Correcciones

- El zoom al 100% despues de recortar conserva la correspondencia con pixeles
  reales.
- La ROI de analisis MTF permanece visible despues de aplicar un recorte.
- Los refrescos interactivos evitan marcar el histograma como pendiente en cada
  paso, pero siguen consolidando metricas exactas al soltar.
- El descargador de actualizaciones omite assets de metadatos y exige SHA-256
  verificable para el instalador.

## Verificacion

- Suite completa: `python -m pytest` -> `414 passed, 2 warnings`.
- Compilacion Python: `python -m compileall src tests`.
- Build Windows: `packaging/windows/build_installer.ps1 -SkipTests
  -StrictExternalTools -RawpyDemosaicWheel <wheel> -RequireAmaze`.
- Smoke tests empaquetados: version CLI, ayuda CLI, herramientas externas,
  C2PA y AMaZE.
