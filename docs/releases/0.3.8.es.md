# ProbRAW 0.3.8

ProbRAW 0.3.8 es una release correctiva para un bug grave de visualizacion y
revelado: lo que se ajusta en pantalla debe coincidir con el TIFF revelado en
aplicaciones con gestion de color. Mantiene el render exacto introducido en
0.3.7 y mejora su coste con cache de demosaico RAW.

## Cambios principales

- La preview RAW usa la misma ruta efectiva de render y color que la
  exportacion, evitando rutas alternativas que pudieran ocultar dominantes o
  ajustes cromaticos que luego si aparecian en el TIFF.
- El demosaico RAW exacto se guarda en una cache persistente del proyecto para
  acelerar recargas, refinados exactos, exportacion individual, lotes y cola de
  revelado sin alterar la receta original.
- La lectura de cache reduce copias completas de memoria en imagenes grandes.
- La cola de revelado retira automaticamente los elementos revelados con exito
  para no volver a procesarlos; los elementos con error permanecen en cola con
  su mensaje.

## Verificacion

- Suite completa: `330 passed, 1 warning`.
- Suite focalizada de cola/sesion: `144 passed`.
- Prueba real de rendimiento en un NEF 8288x5520: primer render exacto 8.56 s,
  segunda carga desde cache 2.30 s con la misma media de imagen.
