# ProbRAW 0.3.5

ProbRAW 0.3.5 es una release de rendimiento y fiabilidad para flujos RAW de
tamaño profesional. Se centra en mantener la interfaz reactiva durante trabajo
pesado de MTF/preview y en hacer que el TIFF final coincida con los ajustes por
archivo.

## Cambios principales

- El analisis MTF frio prepara la ROI a resolucion completa en un proceso
  externo y guarda una cache ROI persistente para recalculos posteriores.
- La barra superior de progreso es ahora el visor global unico para operaciones
  largas de preview, MTF y tareas de fondo, con tiempo transcurrido, duracion
  estimada, tiempo restante y fase.
- La pestaña `Nitidez` ya no duplica una segunda barra local de progreso.
- La carga de preview RAW publica progreso global cuando la duracion esperada u
  observada supera aproximadamente un segundo.
- El revelado de cola aplica ahora la mochila de cada RAW cuando no hay id de
  perfil de ajuste registrado, de modo que nitidez, ruido, aberracion cromatica,
  color y contraste llegan al TIFF final.
- Al cambiar a una imagen sin configurar se restablecen controles de revelado y
  estado ICC activo a la politica neutra ProPhoto/balance-de-camara.
- Los espacios genericos sin carta desactivan profiling mode y balance fijo
  identidad para el render visible/final. La salida RGB de camara sin ICC de
  entrada se rechaza antes de escribir TIFF.

## Verificacion

- Suite completa de tests: `276 passed, 1 warning`.
- Smoke test de flujo real RAW con archivos NEF: carga de preview, cambio de
  imagen, revelado de cola, ICC incrustado en TIFF, efecto real de nitidez y
  verificacion ProbRAW Proof.
- Comprobacion de rendimiento MTF: la preparacion de ROI fria sale del hilo de
  interfaz y el recalculo caliente usa la cache persistente.
