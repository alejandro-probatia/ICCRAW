# ProbRAW 0.3.19

ProbRAW 0.3.19 publica para Windows las mejoras posteriores a 0.3.18 en
preview, empaquetado y gestion de color.

## Preview Y Geometria Del Visor

- La carga RAW puede mostrar una preview embebida provisional con orientacion
  correcta mientras se prepara el render completo.
- El progreso de carga de preview usa estimaciones de tiempo guardadas para
  aperturas lentas.
- Las escrituras pendientes de recorte, rotacion y nivelado del visor en sidecar
  se vacian antes de cambiar de archivo para preservar la geometria durante
  navegacion rapida.
- La gestion de cache de display y preview evita reconstrucciones innecesarias
  manteniendo reproducible el render TIFF final y ProbRAW Proof.

## Empaquetado Y Gestion De Color

- El empaquetado macOS queda disponible desde scripts versionados y documentado
  para validacion, checks AMaZE y firma opcional desde un host macOS.
- El empaquetado Arch/CachyOS sigue documentado con validacion de paquete
  nativo, soporte AMaZE y busqueda de perfiles ICC del sistema.
- La preview ICC usa LittleCMS2 via Pillow `ImageCms`; ArgyllCMS sigue como
  referencia para creacion, validacion y conversiones derivadas de perfiles.

## Verificacion

- Suite completa durante la build del instalador Windows:
  `465 passed, 2 warnings`.
- Check estricto de herramientas externas correcto con ArgyllCMS 3.2.0,
  ExifTool 12.92 y perfiles ICC estandar sRGB/Adobe RGB/ProPhoto.
- Smoke checks empaquetados correctos: `probraw --version`, `probraw --help`,
  `probraw check-tools --strict`, `probraw check-c2pa` y
  `probraw check-amaze`.
- AMaZE activo mediante `rawpy-demosaic 0.10.1`, LibRaw 0.18.7 y
  `DEMOSAIC_PACK_GPL3=True`.
