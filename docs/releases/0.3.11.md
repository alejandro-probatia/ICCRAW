# ProbRAW 0.3.11

ProbRAW 0.3.11 is an interactive-performance release for preview, sharpness
and MTF analysis, while preserving scientific color management with input ICC
profiles and direct conversion to the system monitor profile.

## Main Changes

- Interactive preview again uses bounded proxy sources during color, contrast,
  curve and sharpness drags when the user is not inspecting real pixels.
- At real 100%, sharpness works on the real-resolution viewport when the loaded
  source allows it. If the current display is a proxy, the full source is
  restored before analysis.
- Cached or reduced RAW previews are no longer considered real pixels even if
  the viewer reports 100% scale, avoiding sharpness decisions on a proxy
  version.
- Zoom or viewport changes refresh the visible viewport with active adjustments,
  so the whole viewed image keeps coherent color, contrast and sharpness.
- ESF/LSF/MTF plots update while dragging sharpness controls when the
  full-resolution ROI is hot. The interactive timer throttles instead of
  restarting the calculation on every slider event.
- Documentation consolidates that ProbRAW does not convert display to sRGB:
  images are interpreted through input ICC profiles and displayed through direct
  conversion to the monitor profile configured by the system.
