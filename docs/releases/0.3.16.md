# ProbRAW 0.3.16

ProbRAW 0.3.16 improves TIFF export control and GUI review workflows.

## Main Changes

- TIFF export now supports selectable compression: uncompressed, ZIP/Deflate,
  LZW, JPEG and ZSTD.
- Compressed TIFF writing uses `imagecodecs` and can bound compression threads
  per TIFF from GUI, CLI (`--tiff-workers`) or `PROBRAW_TIFF_MAXWORKERS`.
- Batch export distributes TIFF compression workers across active batch workers
  to use CPU efficiently without oversubscription.
- Horizontal and vertical leveling now use a draggable reference line with live
  angle feedback.
- The folder tree can open a directory in the system file browser from the
  context menu.
- The render queue shows a per-file progress bar while images are processed.

## Fixes

- Exported leveled images crop the valid rotated area and no longer include
  black canvas borders.
- TIFF compression metadata is included in render settings/Proof context.
- Outputs converted through ArgyllCMS `cctiff` are recompressed when a TIFF
  compression mode is selected.
