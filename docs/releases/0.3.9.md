# ProbRAW 0.3.9

ProbRAW 0.3.9 improves the sharpness-analysis workflow and prepares the MTF/CA
code for safer team development.

## Main Changes

- A new lateral chromatic-aberration graph is available next to ESF, LSF and
  MTF. It reuses the selected slanted-edge ROI and displays RGB edge-profile
  differences, CA area, channel crossing shifts and a nearest-neighbour pixel
  strip across the edge.
- ESF is now displayed as a local edge window instead of the full ROI distance
  range, with a pixel-tone strip to inspect how pixel values transition.
- MTF readability is improved with clearer cycles/pixel scaling and standard
  MTF50, MTF30 and MTF10 reference levels.
- MTF/CA analysis classes and helpers include focused annotations for future
  development and review.
