# ProbRAW 0.3.18

ProbRAW 0.3.18 is a hotfix for undoing viewer geometry changes after recropping
or leveling a preview.

## Fixes

- Undo/redo now distinguishes viewer-only geometry changes from changes that
  affect pixel rendering.
- Going back after selecting a crop no longer invalidates the preview cache or
  forces a final preview rebuild when only crop/rotation state changed.

## Arch/CachyOS Packaging

- Package revision `0.3.18-3` adds a native Arch/CachyOS build with an isolated
  venv under `/opt/probraw/venv`, `probraw`/`probraw-ui` launchers,
  conflict/replacement metadata for `iccraw`/`nexoraw` and installed validation
  at `/usr/share/doc/probraw/validate_cachyos_install.sh`.
- The optimized build can compile `rawpy-demosaic` with AMaZE using
  `PROBRAW_ARCH_NATIVE=1` and records hash/metadata for the generated wheel.

## System Color Management

- ICC preview uses LittleCMS2 through Pillow `ImageCms`; ArgyllCMS is reserved
  for `colprof`, `xicclu`/`icclu` and `cctiff`.
- `probraw check-color-environment` documents the display-profile provider,
  Qt/PySide, LittleCMS2, ArgyllCMS, colord, KWin/Wayland and system standard
  profiles.
- Standard profiles are searched in Arch/CachyOS, colord/XDG and Homebrew
  system paths, including `/usr/share/argyllcms/ref`.
