# ProbRAW 0.3.0

ProbRAW 0.3.0 completa el cambio de identidad del proyecto desde el nombre
anterior a ProbRAW. El cambio evita conflicto de marca y alinea el proyecto con
su liderazgo: **Probatia Forensics SL**, en colaboración con la
**Asociación Española de Imagen Científica y Forense**.

## Cambios principales

- Aplicación, documentación, iconos, screenshots, paquete Python y lanzadores
  renombrados a ProbRAW.
- Comandos canónicos: `probraw` y `probraw-ui`.
- Artefactos de release renombrados a `probraw_*` / `probraw-*`.
- Repositorio y comprobador de actualizaciones apuntan a
  `alejandro-probatia/ProbRAW`.
- El paquete Debian declara sustitución/conflicto con `nexoraw` e `iccraw`.

## Compatibilidad

- Las sesiones con `RAW.nexoraw.json` o `RAW.iccraw.json` siguen cargando y se
  migran a `RAW.probraw.json` al guardar.
- Los sidecars `.nexoraw.proof.json` y `.iccraw.proof.json` siguen siendo
  legibles para verificación.
- Las etiquetas C2PA/Proof y variables de entorno beta anteriores se aceptan
  como fallback de migración.
