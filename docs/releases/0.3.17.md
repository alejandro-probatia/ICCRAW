# ProbRAW 0.3.17

ProbRAW 0.3.17 is an interactive-performance maintenance release for repeated
preview and cached RAW session workflows.

## Main Changes

- Lateral chromatic-aberration correction now reuses radial remap coordinates
  for repeated preview edits with the same image size and scale.
- Preview vibrance/saturation adjustments allocate fewer large temporary arrays.
- Demosaic cache pruning is rate-limited after cache writes, reducing repeated
  directory scans during batch sessions.
- Local false-color suppression reuses the output buffer for channel assembly,
  reducing temporary allocations while preserving canonical regression hashes.

## Compatibility

- Canonical TIFF output paths and golden regression hashes are unchanged.
- The sRGB display conversion LUT proposal was not included because it did not
  preserve exact display-buffer bytes in the current test suite.
