# ProbRAW 0.3.3

ProbRAW 0.3.3 publica el nuevo flujo de interfaz centrado en sesión,
calibración, ajustes personalizados y exportación, junto con una política más
clara de previsualización colorimétrica.

## Cambios principales

- `1. Sesión` muestra estadísticas del proyecto y acceso a sesiones recientes.
- `2. Ajustar / Aplicar` organiza la columna derecha en tres pestañas:
  `Color / calibración`, `Ajustes personalizados` y `RAW / exportación`.
- El visor central concentra comparación A/B, aplicación ICC, zoom, 1:1, giro,
  encaje, caché y enfoque de columnas en una barra horizontal con iconos.
- El histograma se mantiene visible en `Ajustes personalizados` y mide la señal
  colorimétrica previa al ICC del monitor.
- Las curvas avanzadas pueden editarse por luminosidad, rojo, verde y azul.
- Al abrir una sesión, `Carta` recupera automáticamente los datos de parches del
  `profile_report.json` guardado.
- Manuales, capturas y documentación de gestión de color actualizados.

## Gestión de color

La visualización sigue esta separación:

1. el perfil ICC de entrada generado con carta interpreta la imagen;
2. la preview se expresa como señal colorimétrica sRGB de revisión;
3. el perfil ICC del monitor del sistema operativo se aplica solo al final para
   mostrar correctamente en pantalla.

El histograma y el overlay de clipping usan el paso 2, no el buffer ya convertido
al monitor. Por eso es importante que el usuario tenga configurado correctamente
el perfil ICC del monitor en el sistema operativo o lo seleccione en la
configuración global de ProbRAW.
