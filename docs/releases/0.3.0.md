# ProbRAW 0.3.0

ProbRAW 0.3.0 completes the project identity change from the previous name to
ProbRAW. The change avoids brand collision and aligns the project with its
leadership: [**Probatia Forensics SL**](https://probatia.com), in collaboration
with the
[**Asociación Española de Imagen Científica y Forense**](https://imagencientifica.es).

## Main Changes

- Application, documentation, icons, screenshots, Python package and launchers
  renamed to ProbRAW.
- Canonical commands: `probraw` and `probraw-ui`.
- Release artifacts renamed to `probraw_*` / `probraw-*`.
- Repository and update checker point to `alejandro-probatia/ProbRAW`.
- Debian package declares replacement/conflict metadata for `nexoraw` and
  `iccraw`.

## Compatibility

- Sessions with `RAW.nexoraw.json` or `RAW.iccraw.json` still load and migrate to
  `RAW.probraw.json` on save.
- `.nexoraw.proof.json` and `.iccraw.proof.json` proof sidecars remain readable
  for verification.
- Previous beta C2PA/Proof labels and environment variables are accepted as
  migration fallbacks.
