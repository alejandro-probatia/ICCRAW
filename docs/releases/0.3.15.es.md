# ProbRAW 0.3.15

ProbRAW 0.3.15 es una release correctiva de rendimiento y fidelidad de
exportacion.

## Cambios Principales

- Los TIFF exportados incluyen ahora la geometria visual de recorte y nivelado
  en la ruta de render.
- Los lotes GUI con varios archivos ya no reutilizan el recorte/nivelado de la
  imagen seleccionada para todos los archivos.
- El muestreo de cartas, el analisis MTF y la cache de preview reducen trabajo
  evitable de memoria e IO.
- La seleccion automatica de workers de batch es menos optimista con RAWs
  comprimidos.
- Los multiplicadores de temperatura/tinte de preview se basan en cromaticidad
  CCT y ratios de blanco en sRGB lineal.

## Correcciones

- Eliminada la ruta experimental LittleCMS/Pillow de 8 bits para TIFF16. La
  conversion ICC de salida sigue en ArgyllCMS `cctiff`.
- El redondeo de recorte exportado sigue ahora la regla `floor/ceil` del visor.
- El perfilado GUI usa por defecto un unico worker en Windows/Qt.
- La cache SHA de RAW puede desactivarse con `PROBRAW_RAW_SHA_CACHE=0` para
  rehash estricto en auditoria.

## Verificacion

- Suite completa: `python -m pytest` -> `426 passed, 2 warnings`.
- Compilacion Python: `python -m compileall src tests`.
- Build Windows: `packaging/windows/build_installer.ps1 -SkipTests
  -StrictExternalTools -RawpyDemosaicWheel <wheel> -RequireAmaze`.
- Smoke tests empaquetados: version CLI, ayuda CLI, herramientas externas,
  C2PA y AMaZE.
