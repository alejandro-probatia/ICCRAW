# ProbRAW 0.3.6

ProbRAW 0.3.6 consolidates per-image adjustment traceability, display color
management and the sharpness/MTF workflow for controlled RAW development.

## Main Changes

- Project-scoped ICC, color/contrast, sharpness and RAW export adjustment
  profiles can be saved, copied, pasted and applied independently per image.
- RAW sidecars and thumbnails now track which ICC, color/contrast, sharpness and
  RAW export adjustments are active for each file.
- The Color/Calibration workflow separates generic ICC profiles, existing camera
  ICC profiles and chart-based ICC generation.
- Display preview uses the selected image ICC as input and the system or
  configured monitor ICC only for screen display.
- RAW/export settings are limited to RAW read and demosaic controls, including
  algorithm-dependent border, 4-color and false-color suppression options.
- Tone-curve histograms update from the adjusted image in real time, show RGB
  columns, and when editing a chromatic channel display only that channel's
  histogram while keeping adjusted RGB curves visible.
- MTF ROI selection no longer reopens the packaged GUI, and Auto Sharpness now
  persists sharpness/radius into the selected RAW sidecar.
- Color/contrast, sharpness/detail and RAW settings autosave to the selected RAW
  sidecar so final TIFF rendering matches the visible adjustment state.
- The user manual and installer guide are updated for the new ICC workflow,
  independent profile categories, thumbnail markers, RGB curves and RAW panel
  limited to reading/demosaicing.

## Verification

- Full test suite: `305 passed, 1 warning`.
- Strict external tools: ArgyllCMS `colprof`, `xicclu`, `cctiff`; ExifTool; C2PA

- Focused GUI tests cover color/contrast sidecar autosave, detail sidecar
  autosave, per-channel tone-curve histograms, MTF auto-sharpness persistence,
  RAW export sidecars and render queue preference for per-file sidecars.
