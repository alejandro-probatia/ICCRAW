# ProbRAW 0.3.1

ProbRAW 0.3.1 is a visual identity release. It keeps the functional changes
from 0.3.0 and replaces the inherited logo with a dedicated ProbRAW mark.

## Main Changes

- New ProbRAW logo with a `P` mark, RAW file geometry and color calibration
  patches.
- New application icon for desktop launchers, Debian packages and Windows
  installers.
- Regenerated SVG, PNG and ICO assets to remove visual leftovers from the
  previous identity.
