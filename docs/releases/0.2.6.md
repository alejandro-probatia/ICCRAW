# ProbRAW 0.2.6

ProbRAW 0.2.6 expands the session ICC workflow with organized profile
management, chart references editable from the interface and visual gamut
diagnostics.

## Highlights

- Advanced ICC profile creation now runs in the background so the application
  does not freeze while the profile is generated.
- Sessions store a catalog of generated or loaded ICC profiles, with explicit
  activation and support for several versions of the same profile.
- The new `Gamut 3D` tab compares profile pairs: session profile, active ICC,
  monitor, sRGB, Adobe RGB, ProPhoto RGB or custom ICC.
- Chart references can be imported, validated, created and edited from a Lab
  table with one color swatch per patch.
- Profiling artifacts are stored per run under
  `00_configuraciones/profile_runs/`, keeping the session history cleaner.
