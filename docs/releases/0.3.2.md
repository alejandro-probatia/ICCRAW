# ProbRAW 0.3.2

ProbRAW 0.3.2 fixes the application icon in Linux menus, especially in Cinnamon
environments where the entry could appear without an icon even though the
hicolor assets were installed.

## Main Changes

- The Debian package `.desktop` entry now uses
  `Icon=/usr/share/pixmaps/probraw.png`.
- The package still ships hicolor icons by size and the scalable SVG, but the
  menu points to an absolute pixmap as a robust fallback.
- Package and installation validations check the real path used by the menu.
