# NexoRAW 0.2.3

Release centrada en coherencia metodologica de gestion de color y trazabilidad
forense de ajustes.

## Cambios principales

- El flujo sin carta ya no genera perfiles `NexoRAW generic ...`.
- sRGB, Adobe RGB (1998) y ProPhoto RGB se tratan como espacios estandar reales:
  LibRaw revela el RAW en el espacio solicitado y NexoRAW incrusta un ICC real
  del sistema o de ArgyllCMS.
- La GUI guarda los ICC estandar de sesion en
  `00_configuraciones/profiles/standard/`.
- `batch-develop --profile` es opcional cuando la receta usa un espacio
  estandar sin carta.
- NexoRAW Proof/C2PA declaran ajustes completos de receta RAW, nitidez/detalle,
  contraste/render y gestion de color. El hash de ajustes queda como control de
  integridad, no como unico dato visible.
- El visor de metadatos muestra los ajustes reproducibles de NexoRAW Proof aun
  cuando se omite la lectura C2PA.

## Resultados locales de referencia

Benchmark half-size con RAW Nikon D850 de 53,9 MB en Windows:

- `camera_raw`: ~0,87 s.
- `srgb`: ~1,32 s, ICC `sRGB.icm`.
- `adobe_rgb`: ~1,24 s, ICC `AdobeRGB1998.icc`.
- `prophoto_rgb`: ~1,21 s, ICC `ProPhoto.icm`.
