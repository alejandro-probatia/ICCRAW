# ProbRAW 0.2.3

Release focused on methodological coherence of color management and traceability
forensic adjustments.

## Main changes

- Letterless flow no longer generates `ProbRAW generic ...` profiles.
- sRGB, Adobe RGB (1998) and ProPhoto RGB are treated as real standard spaces:
  LibRaw reveals the RAW in the requested space and ProbRAW embeds a real ICC
  of the system or ArgyllCMS.
- The GUI saves the standard session ICCs in
  `00_configuraciones/profiles/standard/`.
- `batch-develop --profile` is optional when the recipe uses a space
  standard without letter.
- ProbRAW Proof/C2PA declare full RAW recipe adjustments, sharpness/detail,
  contrast/render and color management. The settings hash remains as a control
  integrity, not as the only visible data.
- Metadata viewer shows reproducible ProbRAW Proof settings even
  when C2PA reading is skipped.

## Reference local results

Half-size benchmark with 53.9 MB Nikon D850 RAW on Windows:

- `camera_raw`: ~0.87s.
- `srgb`: ~1.32 s, ICC `sRGB.icm`.
- `adobe_rgb`: ~1.24 s, ICC `AdobeRGB1998.icc`.
- `prophoto_rgb`: ~1.21 s, ICC `ProPhoto.icm`.
