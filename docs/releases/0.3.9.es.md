# ProbRAW 0.3.9

ProbRAW 0.3.9 mejora el flujo de análisis de nitidez y prepara el código MTF/CA
para desarrollos en equipo más seguros.

## Cambios principales

- Hay un nuevo gráfico de aberración cromática lateral junto a ESF, LSF y MTF.
  Reutiliza la ROI de borde inclinado seleccionada y muestra diferencias de
  perfiles RGB, área CA, desplazamientos de cruce entre canales y una tira de
  píxeles por vecino más próximo a través del borde.
- ESF se muestra ahora como una ventana local del borde en lugar de todo el
  rango de distancias de la ROI, con una tira tonal para inspeccionar cómo
  transicionan los valores de píxel.
- MTF mejora su lectura con una escala ciclos/píxel más clara y niveles estándar
  de referencia MTF50, MTF30 y MTF10.
- Las clases y helpers de análisis MTF/CA incorporan anotaciones focalizadas
  para facilitar futuras revisiones y desarrollo.
