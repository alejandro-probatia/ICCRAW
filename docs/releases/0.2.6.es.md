# NexoRAW 0.2.6

NexoRAW 0.2.6 amplía el flujo ICC de sesión con gestión ordenada de perfiles,
referencias de carta editables desde la interfaz y diagnóstico visual de gamut.

## Puntos principales

- La creación de perfiles ICC avanzados se ejecuta en segundo plano para que la
  aplicación no quede bloqueada durante la generación.
- Las sesiones guardan un catálogo de perfiles ICC generados o cargados, con
  activación explícita y soporte para varias versiones del mismo perfil.
- La nueva pestaña `Gamut 3D` compara pares de perfiles: sesión, ICC activo,
  monitor, sRGB, Adobe RGB, ProPhoto RGB o ICC personalizado.
- Las referencias de carta pueden importarse, validarse, crearse y editarse
  desde una tabla Lab con muestra de color por parche.
- Los artefactos de perfilado se guardan por ejecución en
  `00_configuraciones/profile_runs/`, manteniendo más limpio el historial de
  sesión.
