# ProbRAW 0.3.18

ProbRAW 0.3.18 es una hotfix para deshacer cambios de geometria de visor tras
reencuadrar o nivelar una preview.

## Correcciones

- Deshacer/rehacer distingue ahora cambios solo de geometria de visor de cambios
  que afectan al renderizado de pixeles.
- Volver atras tras seleccionar un recorte ya no invalida la cache de preview ni
  fuerza una reconstruccion final de preview cuando solo cambia el estado de
  recorte/rotacion.

## Empaquetado Arch/CachyOS

- La revision de paquete `0.3.18-3` aĂ±ade build nativa Arch/CachyOS con venv
  aislado en `/opt/probraw/venv`, lanzadores `probraw`/`probraw-ui`, metadatos
  de conflicto/reemplazo de `iccraw`/`nexoraw` y validacion instalada en
  `/usr/share/doc/probraw/validate_cachyos_install.sh`.
- La build optimizada puede compilar `rawpy-demosaic` con AMaZE usando
  `PROBRAW_ARCH_NATIVE=1` y registra hash/metadatos de la wheel generada.

## Gestion de Color del Sistema

- La preview ICC usa LittleCMS2 via Pillow `ImageCms`; ArgyllCMS queda reservado
  para `colprof`, `xicclu`/`icclu` y `cctiff`.
- `probraw check-color-environment` documenta proveedor de perfil de pantalla,
  Qt/PySide, LittleCMS2, ArgyllCMS, colord, KWin/Wayland y perfiles estandar del
  sistema.
- Los perfiles estandar se buscan en rutas de sistema de Arch/CachyOS,
  colord/XDG y Homebrew, incluida `/usr/share/argyllcms/ref`.
