# ProbRAW 0.3.11

ProbRAW 0.3.11 es una release de rendimiento interactivo para preview,
nitidez y analisis MTF, manteniendo la gestion cientifica de color con perfiles
ICC de entrada y conversion directa al perfil de monitor del sistema.

## Cambios principales

- La preview interactiva vuelve a usar fuentes proxy acotadas durante arrastres
  de color, contraste, curvas y nitidez cuando el usuario no esta inspeccionando
  pixeles reales.
- A 100% real, nitidez trabaja sobre el viewport a resolucion real si la fuente
  cargada lo permite. Si la pantalla actual es un proxy, se restaura la fuente
  completa antes de analizar.
- Las previews RAW cacheadas o reducidas ya no se consideran pixeles reales
  aunque el visor indique escala 100%, evitando decisiones de nitidez sobre una
  version proxy.
- Los cambios de zoom o encuadre refrescan el viewport visible con los ajustes
  activos, para que toda la imagen que el usuario ve mantenga color, contraste
  y nitidez coherentes.
- Las graficas ESF/LSF/MTF se actualizan durante el arrastre de controles de
  nitidez cuando la ROI full-res esta caliente. El temporizador interactivo
  aplica throttle en lugar de reiniciar el calculo en cada evento de slider.
- La documentacion consolida que ProbRAW no convierte la visualizacion a sRGB:
  las imagenes se interpretan mediante ICC de entrada y se visualizan mediante
  conversion directa al perfil de monitor configurado por el sistema.
