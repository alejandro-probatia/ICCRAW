# NexoRAW 0.2.5

NexoRAW 0.2.5 is a structural maintenance release focused on making the project
base easier to read, package and grow.

## Highlights

- The canonical implementation package is now `nexoraw`.
- The old internal compatibility namespace has been removed.
- The GUI code is split into smaller modules for session, preview, profile,
  batch, settings and display workflows.
- Tests, scripts, active documentation and installer metadata now use the
  `nexoraw` names consistently.
- New C2PA outputs use `org.probatia.nexoraw.*` labels; verification remains
  compatible with earlier beta outputs that used the previous label namespace.
- The bilingual user manual was expanded with current screenshots, complete GUI
  option reference, full workflows and glossary.
- README, roadmap, color methodology and color pipeline now reflect the active
  ICC-centered workflow; the DCP+ICC roadmap is archived as historical planning.
