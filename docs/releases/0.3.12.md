# ProbRAW 0.3.12

ProbRAW 0.3.12 is a responsiveness release. It restores the smoother preview
cadence from the 0.3.8 line while preserving later fixes for exact histograms,
ICC-managed display, tone-curve interaction and sharpness analysis.

## Main Changes

- Interactive preview refreshes are throttled during drags instead of being
  scheduled on every slider event, preventing preview work from saturating the
  UI event loop.
- ICC profile previews again use bounded sources when the viewer is below 1:1.
  Full-resolution profile preview is still used for real-pixel inspection.
- Viewer histograms and tone-curve histograms remain exact after interaction
  settles; they are not derived from partial viewport patches.
- Tone-curve point drags and black/white curve range sliders avoid expensive
  histogram and overlay work while the pointer is moving, then consolidate once
  on release.
- Sharpness analysis keeps the hot-ROI refresh behavior introduced in 0.3.11,
  including updates when entering the sharpness panel.
- The update assistant can detect the latest GitHub Release, show the selected
  installer, download it to a recognizable user folder and launch it visibly.

## Verification

- Full suite: `python -m pytest` -> `379 passed, 2 warnings`.
- Real RAW interaction check with `AMG_20190709_TRASIEGO_0004.NEF`: preview
  load around 0.54 s; exposure slider events below 6 ms; exact histogram
  available after settling.
- Windows installer artifacts: built and smoke-tested through the packaging
  script before publication.
