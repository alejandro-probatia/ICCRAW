# ProbRAW 0.3.13

ProbRAW 0.3.13 is a bugfix release for a levels/tone-curve histogram
inconsistency found after 0.3.12.

## Bug Notice

In 0.3.12, the upper full-image clipping histogram and the tone-curve editor
histogram could appear inconsistent after changing `Curve white` or `Curve
black`. The upper histogram could still show stale exact metrics while the new
exact real-pixel histogram was pending, and the curve editor histogram could
show the curve output instead of the data entering the curve.

This did not change exported pixel values, but it made the on-screen levels
diagnostic misleading.

## Fixes

- The tone-curve editor histogram now represents the input data entering the
  curve. Moving curve points or the curve black/white range no longer turns the
  plot into a second output histogram.
- The upper RGB histogram now shows an updating state while exact full-image
  clipping metrics are being recalculated.
- Regression tests cover curve input histograms, curve white-point clipping and
  stale exact-histogram state.

## Verification

- Full suite: `python -m pytest` -> `382 passed, 2 warnings`.
- Windows packaged app smoke tests: CLI version, external tools, C2PA and AMaZE.
