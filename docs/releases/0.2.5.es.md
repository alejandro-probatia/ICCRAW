# ProbRAW 0.2.5

ProbRAW 0.2.5 es una release de mantenimiento estructural enfocada en dejar la
base del proyecto mas facil de leer, empaquetar y escalar.

## Puntos principales

- El paquete canonico de implementacion pasa a ser `probraw`.
- Se retira el antiguo namespace interno de compatibilidad.
- El codigo de la GUI queda dividido en modulos mas pequenos para sesiones,
  preview, perfiles, lote, configuracion y visualizacion.
- Tests, scripts, documentacion activa y metadatos de instaladores usan ya de
  forma consistente los nombres `probraw`.
- Las nuevas salidas C2PA usan etiquetas `org.probatia.probraw.*`; la
  verificacion sigue siendo compatible con salidas beta anteriores que usaban el
  namespace previo.
- El manual bilingüe se amplía con capturas actuales, referencia completa de
  opciones de GUI, flujos completos y glosario.
- README, roadmap, metodología de color y pipeline de color reflejan ya el flujo
  activo centrado en ICC; el roadmap DCP+ICC queda archivado como planificación
  histórica.
