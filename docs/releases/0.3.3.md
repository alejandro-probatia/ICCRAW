# ProbRAW 0.3.3

ProbRAW 0.3.3 publishes the new interface flow centered on session management,
calibration, custom adjustments and export, together with a clearer
colorimetric-preview policy.

## Main Changes

- `1. Sesión` shows project statistics and recent-session access.
- `2. Ajustar / Aplicar` organizes the right column into three tabs:
  `Color / calibración`, `Ajustes personalizados` and `RAW / exportación`.
- The central viewer concentrates A/B comparison, ICC application, zoom, 1:1,
  rotation, fit, cache and side-column focus in a compact horizontal toolbar.
- The histogram stays visible in `Ajustes personalizados` and measures the
  colorimetric signal before monitor ICC conversion.
- Advanced curves can be edited by luminance, red, green and blue channels.
- When a session is opened, `Carta` automatically restores patch data from the
  saved `profile_report.json`.
- Manuals, screenshots and color-management documentation were updated.

## Color Management

Display follows this separation:

1. the chart-generated input ICC profile interprets the image;
2. the preview is represented as a colorimetric sRGB review signal;
3. the operating-system monitor ICC is applied only at the end for correct
   on-screen display.

The histogram and clipping overlay use step 2, not the monitor-converted buffer.
For that reason the user must correctly configure the monitor ICC in the
operating system or select it in ProbRAW global settings.
