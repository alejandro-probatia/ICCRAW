# ProbRAW 0.3.13

ProbRAW 0.3.13 es una release de correccion para una incoherencia de
niveles/histograma de curva detectada despues de publicar 0.3.12.

## Aviso Del Bug

En 0.3.12, el histograma superior de clipping de imagen completa y el
histograma del editor de curva podian parecer incoherentes tras cambiar `Blanco
curva` o `Negro curva`. El histograma superior podia seguir mostrando metricas
exactas antiguas mientras el nuevo histograma exacto de pixeles reales estaba
pendiente, y el histograma del editor de curva podia mostrar la salida de la
curva en lugar de los datos que entran en ella.

Esto no cambiaba los pixeles exportados, pero podia hacer enganoso el
diagnostico visual de niveles.

## Correcciones

- El histograma del editor de curva representa ahora los datos que entran en la
  curva. Mover puntos o el rango negro/blanco de curva ya no convierte ese
  grafico en un segundo histograma de salida.
- El histograma RGB superior muestra un estado de actualizacion mientras se
  recalculan las metricas exactas de clipping de imagen completa.
- Las pruebas de regresion cubren histogramas de entrada de curva, clipping por
  blanco de curva y estado antiguo del histograma exacto.

## Verificacion

- Suite completa: `python -m pytest` -> `382 passed, 2 warnings`.
- Smoke tests de app Windows empaquetada: version CLI, herramientas externas,
  C2PA y AMaZE.
