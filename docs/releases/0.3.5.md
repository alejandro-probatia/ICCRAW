# ProbRAW 0.3.5

ProbRAW 0.3.5 is a performance and reliability release for professional-size
RAW workflows. It focuses on keeping the UI responsive during heavy MTF/preview
work and on making final TIFF rendering match the per-file adjustment state.

## Main Changes

- Cold MTF analysis prepares the full-resolution ROI in an external worker
  process and stores a persistent ROI cache for later recalculations.
- The top progress bar is now the single global viewer for long preview, MTF and
  background operations, with elapsed time, estimated duration, remaining time
  and phase.
- The `Nitidez` tab no longer duplicates a second local progress bar.
- RAW preview loading publishes global progress when the expected or observed
  duration is above roughly one second.
- Batch rendering now applies each RAW backpack when no registered development
  profile id is assigned, so sharpening, denoise, chromatic aberration, color
  and contrast settings reach the final TIFF.
- Switching to an unconfigured image resets development controls and active ICC
  state to the neutral ProPhoto/camera-WB policy.
- Generic no-chart output disables profiling mode and identity fixed white
  balance for visible/final rendering. Camera RGB output without an input ICC is
  rejected before writing a TIFF.

## Verification

- Full test suite: `276 passed, 1 warning`.
- Real RAW workflow smoke test with NEF files: preview loading, image switching,
  queue rendering, TIFF ICC embedding, sharpening effect and ProbRAW Proof
  verification.
- MTF performance check: cold ROI preparation moves out of the UI thread and hot
  ROI recalculation runs from the persistent cache.
