# ProbRAW 0.3.1

ProbRAW 0.3.1 es una release de identidad visual. Mantiene los cambios
funcionales de 0.3.0 y sustituye el logo heredado por una marca propia de
ProbRAW.

## Cambios principales

- Nuevo logo ProbRAW con marca `P`, geometria de archivo RAW y parches de
  calibracion de color.
- Nuevo icono de aplicacion para escritorio, paquetes Debian e instaladores
  Windows.
- Regenerados los assets SVG, PNG e ICO para eliminar restos visuales de la
  identidad anterior.
