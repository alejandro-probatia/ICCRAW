# ProbRAW 0.3.15

ProbRAW 0.3.15 is a corrective performance and export-fidelity release.

## Main Changes

- Exported TIFFs now include visual crop and leveling geometry in the render
  path.
- Multi-file GUI batches no longer reuse the selected image crop/leveling
  geometry for every file.
- Chart sampling, MTF analysis and preview disk cache reduce avoidable memory
  and IO work.
- Batch worker auto-selection is less optimistic for compressed RAW files.
- Temperature/tint preview multipliers are based on CCT chromaticity and linear
  sRGB white ratios.

## Fixes

- Removed the experimental 8-bit LittleCMS/Pillow TIFF16 export path. ICC output
  conversion remains on ArgyllCMS `cctiff`.
- Export crop rounding now follows the viewer's `floor/ceil` rule.
- GUI profile generation defaults to a single worker on Windows/Qt.
- RAW SHA cache can be disabled with `PROBRAW_RAW_SHA_CACHE=0` for strict audit
  rehashing.

## Verification

- Full suite: `python -m pytest` -> `426 passed, 2 warnings`.
- Python compilation: `python -m compileall src tests`.
- Windows build: `packaging/windows/build_installer.ps1 -SkipTests
  -StrictExternalTools -RawpyDemosaicWheel <wheel> -RequireAmaze`.
- Packaged smoke tests: CLI version, CLI help, external tools, C2PA and AMaZE.
