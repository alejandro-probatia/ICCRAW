# ProbRAW 0.3.14

ProbRAW 0.3.14 stabilizes the current balance between interactive performance
and accurate GUI readings.

## Main Changes

- Interactive preview updates sampled histogram and levels data during drags
  without blocking the UI, while preserving exact refreshes once interaction
  settles.
- Tone-curve controls keep the editor histogram visible while dragging points
  or the `Curve black`/`Curve white` range.
- The viewer now supports visual crop, horizontal or vertical reference
  leveling, fractional rotation and crop reprojection when preview resolution
  changes or real-pixel view is requested.
- The new `Edit` menu adds undo, redo, clear adjustments and `Ctrl+Z`/`Ctrl+Y`
  shortcuts.

## Fixes

- 100% zoom after crop keeps real-pixel correspondence.
- The MTF analysis ROI remains visible after applying a crop.
- Interactive refreshes avoid marking the histogram as pending at every step,
  while still consolidating exact metrics on release.
- The update downloader skips metadata assets and requires a verifiable
  SHA-256 for the installer.
