# ProbRAW 0.3.19

ProbRAW 0.3.19 publishes the post-0.3.18 preview, packaging and color
management updates for Windows users.

## Preview And Viewer Geometry

- RAW loading can show a correctly oriented provisional embedded preview while
  the full render is prepared.
- Preview load progress uses stored timing estimates for slower opens.
- Pending viewer crop, rotation and leveling sidecar writes are flushed before
  file changes so geometry is preserved during fast navigation.
- Display and preview cache handling avoids unnecessary rebuilds while keeping
  final TIFF rendering and ProbRAW Proof output reproducible.

## Packaging And Color Management

- macOS packaging is available from versioned scripts and documented for
  validation, AMaZE checks and optional signing from a macOS host.
- Arch/CachyOS packaging remains documented with native package validation,
  AMaZE support and system ICC profile discovery.
- ICC preview uses LittleCMS2 through Pillow `ImageCms`; ArgyllCMS remains the
  reference for profile creation, validation and derived conversions.

## Verification

- Full suite during the Windows installer build: `465 passed, 2 warnings`.
- Strict external tools check passed with ArgyllCMS 3.2.0, ExifTool 12.92 and
  standard sRGB/Adobe RGB/ProPhoto ICC profiles.
- Packaged smoke checks passed: `probraw --version`, `probraw --help`,
  `probraw check-tools --strict`, `probraw check-c2pa` and
  `probraw check-amaze`.
- AMaZE is active through `rawpy-demosaic 0.10.1`, LibRaw 0.18.7 and
  `DEMOSAIC_PACK_GPL3=True`.
