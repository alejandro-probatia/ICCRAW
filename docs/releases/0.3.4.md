# ProbRAW 0.3.4

ProbRAW 0.3.4 focuses on the new sharpness-analysis workflow: persistent
full-resolution slanted-edge MTF measurement and visual comparison of saved
curves.

## Main Changes

- The `Nitidez` tab stores MTF metrics plus `ESF`, `LSF` and `MTF` curves in
  each RAW sidecar.
- Reopening an image restores the saved ROI and curves, so the edge does not
  need to be selected again before recalculating.
- Recalculation loads the real source image at full resolution and maps the
  viewer ROI to analysis coordinates instead of measuring a thumbnail or reduced
  preview.
- The MTF comparison dialog can overlay saved curves from two selected
  thumbnails and keeps the numeric metric table.
- Pixel-pitch and lp/mm values are tied to the analysis dimensions, EXIF
  metadata or the manual sensor-size controls.
- The English Qt catalog and user manuals were updated for the new analysis and
  comparison tools.

## Measurement Note

The MTF curves are calculated from the full-resolution image data available to
the selected file and current recipe. The sidecar stores both analysis ROI and
viewer ROI so later previews can restore the overlay correctly even when their
display size changes.
