# ProbRAW 0.3.17

ProbRAW 0.3.17 es una release de mantenimiento de rendimiento interactivo para
flujos repetidos de preview y sesiones RAW con cache.

## Cambios Principales

- La correccion de aberracion cromatica lateral reutiliza coordenadas de remapeo
  radial en ediciones repetidas de preview con el mismo tamano y escala.
- Los ajustes de vibrance/saturacion de preview asignan menos arrays temporales
  grandes.
- La poda de cache de demosaic queda limitada por tiempo tras escrituras de
  cache, reduciendo escaneos repetidos de directorio durante sesiones batch.
- La supresion local de falso color reutiliza el buffer de salida para ensamblar
  canales, reduciendo asignaciones temporales y preservando los hashes canonicos
  de regresion.

## Compatibilidad

- Las rutas canonicas de salida TIFF y los hashes golden de regresion no cambian.
- La propuesta de LUT para conversion sRGB de display no se incluye porque no
  preservaba los bytes exactos del buffer de display en la suite actual.

- Suite completa: `python -m pytest tests` -> `438 passed, 2 warnings`.
- Suite de regresion: `python -m pytest tests/regression` -> `2 passed`.
- Suite enfocada preview/pipeline: `pytest tests/test_preview.py

- Compilacion Python: `python -m compileall src/probraw`.
- Smoke benchmark GUI: `scripts/benchmark_gui_interaction.py --synthetic-size
