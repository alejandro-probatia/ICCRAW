# GitHub issues to create (migration from `docs/ISSUES.md`)

This file translates each P0-P3 entry into a working issue on GitHub.

- Canonical source: `docs/ISSUES.md`.
- Acceptance criteria: derived from `docs/OPERATIVE_REVIEW_PLAN.md`.
- Associated script: `scripts/create_github_issues.sh`.

Format per issue:

- Suggested title (imperative and actionable).
- Tags: priority + area + type.
- Marked `good-first-issue` when applicable.
- Suggested body to paste on GitHub.
